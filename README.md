# PGATS - CI - MOD04

Projeto de automação de testes, contendo testes unitários com geração de relatório com Mochawesome e integração com Pipeline do GitHubActions

## Pré-requisitos

1. Instale o [git](https://git-scm.com)
2. Instale o [nodejs](https://nodejs.org/)
3. Clone o repositório para sua máquina
4. Instale as dependências
   ```shell
   cd pgats-ci-conclusao-mod04
   npm install
   ```
5. Execute os testes de unidade - isso vai gerar um relatório
   ```shell
   npm test-report
   ```
6. Abra o relatório de cobertura de código em `mochawesome-report`
